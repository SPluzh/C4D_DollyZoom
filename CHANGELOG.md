# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2026-02-28
### Added
- Integer focal length adjustment when holding the **Control (Ctrl)** key.
- Custom icon for the Dolly Zoom tool.

## [1.0.1] - 2026-02-27
### Added
- Support for geometric components (Points, Edges, Polygons) as pivot targets.

## [1.0.0] - 2026-02-25
### Added
- Initial release of the Dolly Zoom (Vertigo effect) tool for Cinema 4D.
